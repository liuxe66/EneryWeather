# 活力天气

### 前言

主要为了学习kotlin和jetpack

在网上找的[魅族天气API](https://blog.csdn.net/qq_24810411/article/details/89175825)

项目中主要使用kotlin+LiveData+ViewModel+协程+retrofit2完成
 
### 主要功能

+ 实时天气+每三小时天气
+ 未来一周天气
+ 生活指数
+ 定位+选择城市
